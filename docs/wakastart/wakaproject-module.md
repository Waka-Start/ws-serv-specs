# WakaProject dans WakaStart

> **État** : document de cadrage initial. À étendre au fil de la définition fonctionnelle du module.

Le module **WakaProject** est le module de gestion de projet intégré à WakaStart.

## Rattachement organisationnel

Les différents projets gérés dans WakaProject sont **associés au niveau organisation**, car ils peuvent être créés **avant même que la WakaApp ne soit créée**.

La visibilité des projets pour les administrateurs métiers doit être liée à l'appartenance à l'organisation, en particulier pour les porteurs de projet.

La liste de projets présentée doit être liée à une organisation au sens WakaStart.

## Gestion des droits

Une gestion de droits spécifique pour la gestion de projet est décrite dans la documentation. Elle permet de créer des **profils externes** pour des administrateurs clients qui peuvent être :

- **porteurs de projet**,
- **testeurs de projet**.

## Création des projets

Seuls les **superadministrateurs Waka** peuvent créer de nouveaux projets — ce n'est **pas** le cas des administrateurs d'organisation.
