# Spécifications WakaStart — Référentiel des pratiques

> **État** : document de cadrage en cours de remplissage par les équipes internes. Les chapitres détaillés seront ajoutés progressivement par les contributeurs identifiés ci-dessous.

## Objectif

Rassembler toutes les informations techniques sur la plateforme WakaStart afin de pouvoir réaliser des analyses et des PRD de manière fiable, incluant toutes les spécifications internes.

Ce document vise à regrouper toutes les informations sur les bonnes pratiques Waka issues des différentes personnes intervenantes :

- création de WakaStart,
- infrastructure,
- conception,
- outils en amont de génération des spécifications.

L'objectif final : produire un PDF unique qui servira à la fois pour la création des spécifications (incluant toutes les bonnes pratiques Waka) et pour le codage lui-même (PRD et agents Claude).

## Public cible

**Les destinataires sont les IA, pas les humains.** Les éléments doivent être très techniques et très concis (pas de blabla).

Une fois consolidés, les éléments seront transformés :
- en **Gamma** pour la partie visuelle et commerciale,
- en **spécifications de PRD** et **agents Claude** pour la partie technique.
